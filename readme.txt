Plugin criado para integrar aplicações scriptcase em páginas ou posts no WordPress.

== Description ==

Plugin criado para integrar aplicações scriptcase em páginas ou posts no WordPress isso é feito utilizando shortcode. Você pode definir os parâmetros largura e altura onde será exibida sua aplicação. Exemplo de uso: [scriptcase width="600" height="300" url="http://www.seudominio.com.br/aplicacao_scriptcase"].Na versão 0.1.0 já é possível usar o atalho pelo TinyMCE.

Os valores default das propriedades são:
scrolling = no, width = 100%, height = 500,frameborder = 0, marginheight = 0, style = border:0,

Você pode alterar de acordo com as suas necessidades como no exemplo abaixo:
[scriptcase width="600" height="300" scrolling = "yes" frameborder = "2" url="http://www.seudominio.com.br/aplicacao_scriptcase"]



== Installation ==

Instalação Automática 

Para instalar é muito simples:

1. No admin do wordpress clique em plugins no menu lateral esquerdo, depois clique adionar novo.
2. No campos pesquisar digite wp-scriptcase e clique pesquisar plugins.
3. Depois clique instalar agora.
4. Após instalação o plugin será ativado.Aproveite.

Instalação Manual

O método de instalação manual envolve o download nosso plugin e enviá-lo para seu servidor através de seu aplicativo FTP favorito.

1. Baixe o arquivo de plug-in para o seu computador e descompactá-lo
2. Usando um programa de FTP, ou o painel de controle de hospedagem, fazer o upload do plugin pasta descompactada para wp-content/plugins / diretório de sua instalação WordPress.
3. Ative o plugin no menu Plugins dentro da administração do WordPress.

== Screenshots ==

1. - Ao ativar o plugin um botão irá aparecer na barra do editor de páginas ou posts.
2. - Ao clicar no botão irá aparecer uma caixa para inserir os dados da aplicação depois de preenchido clique em inserir.
3. - Acesso o seu website e veja a aplicação na página onde selecionou para ser inserida a aplicação.
